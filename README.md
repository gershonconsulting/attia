# ATTIA.COM - Unite the Attia Family Worldwide

![Attia Logo](https://www.attia.com/AttiaLogo.png)

**Version:** 1.0.0  
**Published:** 2026-04-19

A modern, professional website dedicated to uniting the Attia family worldwide. Created by **Olivier Attia** to build a global community around the proud Attia name.

## 🌟 Project Overview

**Attia.com** is a community-driven platform that aims to connect people with the Attia surname from around the world. The website offers free @attia.com email addresses to anyone with the Attia surname, helping to build a strong, global network.

### Mission Statement
> *"This name is strong. This name is powerful. Let's make it stronger together."*

## 🎨 Design Features

### Color Scheme
- **Primary Blue**: `#0033CC` (Deep, professional blue from the logo)
- **Accent Blue**: `#3366FF` and `#4D79FF` (Interactive elements)
- **Light Blue Gradients**: `#E3F2FD`, `#BBDEFB`, `#90CAF9` (Soft, professional backgrounds)
- **Neutral Tones**: Light grays (`#f7f9fc`, `#fbfcfd`) for subtle, clean backgrounds
- **Text Colors**: Professional dark grays for excellent readability

### Typography
- **Font Family**: Inter (Google Fonts)

- Clean, modern, highly readable across all devices

- Light, airy professional design with subtle gradients
